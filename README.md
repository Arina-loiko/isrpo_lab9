Изучить принципы адаптивной верстки и медиа-запросов - создать адаптивную страницу портфолио, которая корректно отображается на всех устройствах

## Структура проекта

```
isrpo_lab9/
├── index.html            - портфолио (базовая страница)
├── styles.css            - стили для портфолио
├── responsive.html       - адаптивный макет с Flexbox и Grid
├── responsive.css        - стили адаптивного макета
├── nav-practice.html     - самостоятельное задание 1: адаптивная навигация
├── flexbox-practice.html - самостоятельное задание 2: Flexbox макет
├── README.md             - описание проекта
└── img/                  - скриншоты
```

## Файлы

### index.html + styles.css
Страница портфолио с секциями: шапка, hero, навыки, проекты, футер
Адаптируется под мобильные, планшеты и десктоп через медиа-запросы

### responsive.html + responsive.css
Учебный адаптивный макет по заданию
Breakpoints:
- мобильный (по умолчанию): колонки в столбик, меню вертикальное
- планшет (768px+): меню горизонтальное, features в ряд, cards 2 колонки
- десктоп (992px+): cards 3 колонки, max-width 1200px
- большой десктоп (1200px+): центрирование через padding

### nav-practice.html
Адаптивное меню - на мобильных вертикально (flex-direction: column), на 768px+ горизонтально (flex-direction: row) с hover-эффектами

### flexbox-practice.html
Flexbox макет - header с логотипом слева и навигацией справа, секция "О нас" с тремя колонками, footer по центру

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Mobile First подход

## Breakpoints

| Устройство | Ширина |
|---|---|
| Мобильный | < 768px |
| Планшет | >= 768px |
| Десктоп | >= 992px |
| Большой десктоп | >= 1200px |
